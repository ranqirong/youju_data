# ERP 新房合同报表生成工具使用说明

## 📋 工具概述

`generate-report.py` 是一个统一的报表生成工具，支持：
1. **每日数据报表**：使用指定某天导出的数据生成
2. **每周数据报表**：按指定周时间段导出后生成

## 🚀 使用方式

### 方式 1: 从 API 导出数据并生成报表

#### 生成每日报表

```bash
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --export-daily \
  --date 2026-03-26
```

**说明**:
- 调用 ERP API 导出指定日期（2026-03-26）的新房合同数据
- 自动生成每日数据报表
- 输出文件：`~/Desktop/ERP 导出/南宁新房/南宁新房每日数据_20260326.xlsx`

#### 生成每周报表

```bash
# 使用当前周
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --export-weekly

# 或指定周时间段
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --export-weekly \
  --week-start 2026-03-23 \
  --week-end 2026-03-29
```

**说明**:
- 调用 ERP API 导出指定周时间段的新房合同数据
- 自动生成每周数据报表
- 输出文件：`~/Desktop/ERP 导出/南宁新房/南宁新房每周数据_20260323.xlsx`

---

### 方式 2: 从已有 Excel 文件生成报表

如果您已经有导出的 Excel 文件，可以直接使用：

#### 生成每日报表

```bash
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --generate-daily \
  --input "/Users/lichunwei/Desktop/ERP 导出/新房合同_20260326_235330.xlsx" \
  --date 2026-03-26
```

**参数说明**:
- `--input`: 输入的 Excel 文件路径
- `--date`: 要生成报表的日期（过滤该日期的数据）

#### 生成每周报表

```bash
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --generate-weekly \
  --input "/Users/lichunwei/Desktop/ERP 导出/新房合同_20260326_235330.xlsx" \
  --week-start 2026-03-23 \
  --week-end 2026-03-29
```

**参数说明**:
- `--input`: 输入的 Excel 文件路径
- `--week-start`: 周开始日期（周一）
- `--week-end`: 周结束日期（周日）

## 🔧 常用示例

### 示例 1: 生成昨天的每日报表

```bash
# 获取昨天的日期
YESTERDAY=$(date -d "yesterday" +%Y-%m-%d)

python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --export-daily \
  --date $YESTERDAY
```

### 示例 2: 生成本周的每周报表

```bash
# 自动计算本周一
WEEK_START=$(date -d "monday" +%Y-%m-%d)

python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --export-weekly \
  --week-start $WEEK_START
```

### 示例 3: 从指定文件生成报表

```bash
# 从手动导出的 Excel 文件生成每日报表

  --generate-daily \
  --input "~/Desktop/ERP 导出/新房合同_20260327_091346.xlsx" \
  --date 2026-03-27

  --generate-weekly \
  --input "~/Desktop/ERP 导出/新房合同_20260327_091346.xlsx" \

  --week-end 2026-03-30
```

## 🔐 凭证配置

首次使用需要配置 ERP 登录凭证。凭证会保存在配置文件中，后续使用无需重复配置。

如果提示未找到凭证，运行：

```bash
python3 ~/.openclaw/skills/erp-api-login/scripts/setup-credentials.py
```

按提示输入手机号和密码即可。

## 📊 数据说明

### 日期格式

工具支持多种日期格式自动转换：
- `2026/03/26`
- `2026-03-26`
- `2026/03/26 00:00:00`
- `2026-03-26 00:00:00`

### 周计算规则

- 周一为一周的开始
- 周日为一周的结束
- 自动计算给定日期所在周的起止日期

### 数据过滤

- **每日报表**: 只显示指定日期的数据
- **每周报表**: 只显示指定周时间段内的数据

## 🐛 故障排除

### 问题 1: 提示"未找到 ERP 登录凭证"

**解决**: 运行凭证配置脚本
```bash
python3 ~/.openclaw/skills/erp-api-login/scripts/setup-credentials.py
```

### 问题 2: 导出的数据为空

**原因**: 指定日期/时间段内没有新的合同数据

**解决**: 
- 检查日期是否正确
- 尝试更大的时间范围
- 确认 ERP 系统中有对应数据

### 问题 3: 报表中某些字段为空

**原因**: 原始导出的 Excel 中这些字段本身就没有数据

**解决**: 这是正常现象，表示该记录在这些维度上没有数据
