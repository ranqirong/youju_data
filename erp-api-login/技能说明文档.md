# ERP API Login 技能 - 完整功能说明

## 📋 技能概述

**ERP API Login** 是一个基于 API 的 ERP 系统登录和数据导出工具，专为房产评估产品设计。通过调用 SSO 认证接口链获取 Token，然后调用各种业务接口导出数据。

**版本**: v20260327-final  
**适用场景**: 新房合同数据导出、报表生成、竞对数据分析、带看统计等

---

## 🔐 认证流程

```
1. 访问登录页面       → 获取 Cookie
2. 获取 RSA 公钥      → 【公钥接口】
3. 获取 code_key     → 【授权接口】
4. faceLogin         → 【人脸登录接口】
5. execute           → 【密码登录接口】
6. accessToken       → 【令牌接口】
7. 业务接口调用      → Authorization: <token> (不加 Bearer!)
```

**关键发现**:
- Authorization header **不需要** "Bearer " 前缀
- 直接使用 Token 即可
- Token 有效期约 12 小时

## 📊 核心功能

### 1. 新房合同导出与报表生成 ⭐⭐⭐

#### 功能说明
- 导出新房合同数据（按日期范围）
- 生成每日数据报表（13 个字段）
- 生成每周数据报表（11 个字段）

#### 输出字段（每周报表）
| 字段 | 说明 | 计算方式 |
|------|------|----------|
| 认购套数 | 认购套数 | 统计认购状态记录数 |
| 认购业绩 | 认购金额 | 基础应付金额求和 |
| 签约套数 | 签约套数 | 统计签约状态记录数 |
| 签约业绩 | 签约金额 | 基础应付金额求和 |
| 套数合计 | 总套数 | 认购套数 + 签约套数 |
| 退单套数 | 退单套数 | 统计退单/作废状态记录数 |
| 退单业绩 | 退单金额 | 基础应付金额求和 |
| 剔除退单套数 | 净套数 | 套数合计 - 退单套数 |
| 剔除退单业绩 | 净业绩 | 认购业绩 + 签约业绩 - 退单业绩 |

#### 使用示例
```bash
# 生成每日报表
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --export-daily --date 2026-03-27

# 生成每周报表
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --export-weekly --week-start 2026-03-24 --week-end 2026-03-30

# 从已有 Excel 生成
python3 ~/.openclaw/skills/erp-api-login/scripts/generate-report.py \
  --generate-daily \
  --input "~/Desktop/ERP 导出/新房合同_xxx.xlsx" \
  --date 2026-03-27
```

#### 相关脚本
- `scripts/generate-report.py` - 主入口
- `scripts/process-new-house-data-v4.py` - 数据处理（最新版）

| 物业类型 | 住宅/商业/办公等 |
| 项目等级 | A、A-、B、B+ 等 |
| 售卖状态 | 在售/待售/售罄/停售 |
| 均价区间 | 价格范围 |
| 面积段 | 面积范围 |
| 装修状况 | 装修情况 |
| 交房时间 | 交房日期 |
| 项目经理 | 项目经理姓名 |
| 驻场 | 驻场人员姓名 |

### 5. 带看进场统计（含周环比、月环比） ⭐⭐⭐

#### 功能说明
根据**首看时间**统计进场量，包含周环比和月环比分析。分一级项目（A 级）和二级项目（A-、B、B+ 级）。

#### 输出字段
| 字段 | 说明 | 计算方式 |
|------|------|----------|
| 本周进场量 | 本周（周一 - 周日）进场量 | 首看时间在本周内 |
| 上周进场量 | 上周进场量 | 首看时间在上周内 |
| 本周 - 上周 | 周增减量 | 本周 - 上周 |
| 周环比 | 周增长率 | (本周 - 上周) / 上周 × 100% |
| 本月进场量 | 本月进场量 | 首看时间在本月内 |
| 上月进场量 | 上月进场量 | 首看时间在上月内 |
| 本月 - 上月 | 月增减量 | 本月 - 上月 |
| 月环比 | 月增长率 | (本月 - 上月) / 上月 × 100% |

#### 项目分类
| 类型 | 楼盘等级 |
|------|---------|
| 一级项目 | A |
| 二级项目 | A-、B、B+ |

#### 使用示例

### 6. 竞对数据导出 ⭐ NEW

#### 功能说明
导出竞争对手对比数据，支持按项目或按区域统计。异步导出，自动获取文件列表。

#### 接口信息
- **接口**: `/api/common.stats.api/rival/stats/pcPkExport`
- **方法**: POST
- **输出**: 异步导出任务

#### 输入参数
| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| `summaryType` | int | 统计类型 (1: 按项目，2: 按区域) | 1 |
| `deptIds` | array | 部门 ID 列表 | ['625864877560328320'] |
| `rivalIds` | array | 竞对公司 ID 列表 | [] |
| `startInclusive` | long | 开始时间戳（毫秒） | 30 天前 |
| `endExclusive` | long | 结束时间戳（毫秒） | 今天 |
| `fileName` | string | 导出文件名 | '竞对数据导出' |

#### 使用示例
```bash
# 导出本月竞对数据
python3 ~/.openclaw/skills/erp-api-login/scripts/export-rival-stats.py \
  --start-date 2026-03-01 --end-date 2026-03-31

# 按区域统计
python3 ~/.openclaw/skills/erp-api-login/scripts/export-rival-stats.py \
  --summary-type 2 \
  --start-date 2026-03-01 --end-date 2026-03-31

# 指定竞对公司
python3 ~/.openclaw/skills/erp-api-login/scripts/export-rival-stats.py \
  --rival-ids 1039119892642533376 1004690592153739264 \
  --start-date 2026-03-01 --end-date 2026-03-31
```

#### 相关脚本
- `scripts/export-rival-stats.py`
- `scripts/erp-export.py` (包含 `export_rival_stats()` 和 `get_export_file_list()` 方法)

## ⚠️ 注意事项

1. **凭证配置**: 首次使用需要配置 ERP 登录凭证（手机号和密码）
   ```bash
   python3 ~/.openclaw/skills/erp-api-login/scripts/setup-credentials.py
   ```

2. **带看数据导出**: 大数据量可能超时（504 Gateway Time-out），建议：
   - 分周导出（每 7 天一次）
   - 减少每页数量（page_size=200）
   - 在网络状况良好时重试

3. **日期范围**: 本周报表自动计算当前周的周一到周日

4. **数据格式**: 支持 `2026/03/26` 和 `2026-03-26` 两种日期格式

5. **业绩计算**: 使用**基础应付金额**而非认购/签约总价

6. **项目分类**:
   - 一级项目：楼盘等级 A
   - 二级项目：楼盘等级 A-、B、B+

7. **竞对数据导出**: 异步导出，创建任务后需等待几秒再获取文件列表
